# 기능 목록

## 로또 생성 [V]
- 1 ~ 45까지 중복되지 않는 숫자 6개를 랜덤으로 생성

## 번호 입력 유효성 검사[V]
- 구매 번호는 6개
- 구매 번호는 1 ~ 45까지의 숫자
- 구매 번호는 서로 다른 숫자
- 보너스 번호는 1 ~45 까지의 숫자
- 보너스 번호는 구매 번호와 다른 숫자

## 금액 입력 [V]
- 1,000원으로 나누어 떨어지지 않는 경우 예외 처리

## 등수 계산
- 숫자 일치 등수별 당첨금액 구현

## 당첨 검사
- 로또가 몇개 맞았는지 갯수 세고 등수로 치환

## 계산
- 수익률 계산 후 소수점 둘째 자리에서 반올림

## 입출력
### 입력
- 단순 기본 메시지와 입력받기
### 출력
- 구입한 복권 출력 메시지
- 통계 출력 메시지들 (등수 및 수익률)
